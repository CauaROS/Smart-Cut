## Smart-Cut
É uma aplicação simples que serve para o corte automático de vídeos curtos utilizando apenas o processamento do sistema, aplicação não armazena os vídeos apenas faz o recorde e reprocessamento de forma local (utilizando o hardware do usuário).

## Rodar Localmente 
**Pré-requisitos:**  Node.js

Node.Js 
https://nodejs.org/pt-br/download

Após a instalação do NodeJs
Na pasta do arquivo abra o terminal e execute os seguintes comandos:
1. Instalar dependências:
   `npm install`
3. Rodar aplicação:
   `npm run dev`
